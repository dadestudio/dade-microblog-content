# dade-microblog-content

Source + archive for [dade.micro.blog](https://dade.micro.blog), mirrored from Micro.blog. (Future custom domain: blog.dade.studio, deferred.)

- `posts/` - published posts (mirrored from Micro.blog)
- `drafts/` - work in progress (gitignored)
- `pages/` - static pages (About, Now, Projects, Colophon)
- `assets/` - images

Micro.blog writes to this repo automatically. Local edits require confirming whether Micro.blog supports two-way sync (see Phase 2.3.3).
